# Password Strength Checker

A simple web application built with Python and Flask that checks the strength of passwords. This application evaluates passwords based on various criteria and provides feedback to users on how to improve their password strength.

## Features

- Check password strength based on length, complexity, and common patterns.
- User-friendly web interface.
- Real-time feedback on password strength as the user types.
- Suggestions for creating stronger passwords.

## Technologies Used

- Python
- Flask
- HTML


## Installation

### Prerequisites

- Python 3.x
- pip (Python package installer)

### Steps to Run the Application
python app.py
http://127.0.0.1:5000
Usage
1.Enter a password in the input field on the main page.
2.The application will evaluate the password strength and provide feedback.
3.Follow the suggestions to create a stronger password.
